# ⬡ HashPilot: Time-Bending Strategy Suite

HashPilot is a production-grade, retro-futuristic "Mining Intelligence Terminal" built exclusively for the Club HashCash ecosystem on Avalanche. It combines real-time blockchain data with advanced AI modeling to provide miners with tactical decision support.

 🛠 Features

 1. 🔮 The Claim Oracle (AI Analytics)
The Oracle analyzes your current mining status (Daily hCASH, claimed rewards, etc.) and provides a definitive "WAIT vs. CLAIM" verdict. It uses custom heuristics and AI cross-verification to help miners optimize gas and timing.

 2. 🛡 Miner Archetypes (Identity)
A deep-dive personality classifier for the HashCash community.
- Interactive assessment to find your mining persona (The Archer, The Vault, The Catalyst, etc.).
Dynamic Identity Badges: Exportable, high-fidelity profile cards with localized stats.
3-Move Playbooks: Tailored strategies based on your specific risk profile.

 3. 🚨 The FOMO Machine (Counterfactual Modeling)
A reality-distorted simulator that calculates the "Cost of Hesitation."
- Opportunity Loss Tracking: See exactly how much hCASH you missed by starting late or staying under-optimized.
- High-Intensity Visualization: Glitch-aesthetic reports that can be downloaded as field archives.

 4. 📂 Tactical Dashboard
A command center displaying live hCASH market prices (via DEX Screener), network status, and claim cycle progress.



 💻 Tech Stack
- Framework: Next.js 14 (App Router)
- Styling: Tailwind CSS + Framer Motion (Glitch & Terminal animations)
- Web3: Wagmi / Viem / RainbowKit (Avalanche Network)
- AI Backend: Anthropic Claude 3.5 Sonnet (Tactical Comms)
- Data Layers: DEX Screener API (hCASH Price), CoinGecko (AVAX Price)
- Image Generation: html-to-image (Badge & Report exports)



 🛠 Development & Setup

 Prerequisites
- Node.js 18+
- [Anthropic API Key](https://console.anthropic.com/) (for Strategy Comms)

 Installation
bash
# Clone the repository
git clone https://github.com/natureloved/HashPilot.git

# Install dependencies
npm install

# Setup Environment Variables
# Create a .env.local file with:
# ANTHROPIC_API_KEY=your_api_key_here

# Run the terminal
npm run dev
